Uma tabela é constituída por uma tag de <table>, a qual possui outras tags em seu interior. Toda tag que for contida dentro de uma table, também deve estar contida dentro de uma linha de tabela, representada pela tag <tr>. Dentro das linhas de tabela, podem ser posicionados vários dados de tabela - tag <td>. Além disso um cabeçalho de tabela - tag <th> - pode ficar no topo da tabela, dividindo os dados da tabela em colunas que seguem os cabeçalhos.
